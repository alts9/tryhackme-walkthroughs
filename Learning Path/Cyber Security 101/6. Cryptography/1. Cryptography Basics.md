##### Task 6: Basic Math
1. What’s `1001 ⊕ 1010`?
	- `⊕` or `XOR` or `exclusive OR` only return 1 if both bits are different
	- Bits 1: `1⊕1 = 0`
	- Bits 2: `0⊕0 = 0`
	- Bits 3: `0⊕1 = 1`
	- Bits 4: `1⊕0 = 1`
	- Answer: `0011`
2. What’s `118613842%9091`?
	- We can use terminal with `$(()))` to make it evaluated as math expression
	- `echo $((118613842 % 9091))`
		- ![](../../../Attachment/2026-04-13_03-19-34_edit.png)
	- Answer: `3565`
3. What’s `60%12`?
	- `echo $((60 % 12))` 
		- ![](../../../Attachment/2026-04-13_03-21-05_edit.png)
	- Answer: `0`
